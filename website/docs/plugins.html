PLUGINS

Extend Loopwork with composable plugins

Loopwork uses a Next.js-style composable plugin architecture. Each plugin is a function that returns configuration enhancements.

Notification Plugins

Telegram

Get task completion and failure notifications via Telegram bot.

Built-in Notification
typescript 
import { withTelegram } from 'loopwork/plugins'

withTelegram({
  botToken: process.env.TELEGRAM_BOT_TOKEN,
  chatId: process.env.TELEGRAM_CHAT_ID,
  notifyOnComplete: true,
  notifyOnFail: true,
})

Discord

Send notifications to Discord channels via webhooks.

Built-in Notification
typescript 
import { withDiscord } from 'loopwork/plugins'

withDiscord({
  webhookUrl: process.env.DISCORD_WEBHOOK_URL,
  notifyOnComplete: true,
  notifyOnFail: true,
})

Project Management Plugins

Asana

Integration Sync

Cost Tracking

Cost Tracking

Monitor token usage and enforce daily budgets to control AI spending.

Monitoring Budget
typescript 
import { withCostTracking } from 'loopwork/plugins'

withCostTracking({
  enabled: true,
  dailyBudget: 10.00,  // Stop if over $10/day
})

Git Integration

Git Auto-Commit

Automatically commit changes after each successful task completion.

Git Automation
typescript 
import { withGitAutoCommit } from 'loopwork/plugins'

withGitAutoCommit({
  enabled: true,
  addAll: true,
  coAuthor: 'Loopwork AI <noreply@loopwork.ai>',
  skipIfNoChanges: true,
})

Dynamic Tasks

Dynamic Task Creation

Automatically create follow-up tasks based on AI output analysis.

AI Automation
typescript 
import { withDynamicTasks } from 'loopwork/plugins'

withDynamicTasks({
  enabled: true,
  analyzer: 'pattern',  // or 'llm' for smarter analysis
  createSubTasks: true,
  maxTasksPerExecution: 5,
  autoApprove: true,
})

Complete Example

typescript 
import { compose, defineConfig } from 'loopwork'
import { withJSONBackend } from 'loopwork/backends'
import { 
  withTelegram, 
  withCostTracking, 
  withGitAutoCommit,
  withDynamicTasks 
} from 'loopwork/plugins'

export default compose(
  withJSONBackend({
    tasksFile: '.specs/tasks/tasks.json'
  }),
  withTelegram({
    botToken: process.env.TELEGRAM_BOT_TOKEN,
    chatId: process.env.TELEGRAM_CHAT_ID,
    notifyOnComplete: true,
    notifyOnFail: true,
  }),
  withCostTracking({
    enabled: true,
    dailyBudget: 10.00,
  }),
  withGitAutoCommit({
    enabled: true,
    addAll: true,
  }),
  withDynamicTasks({
    enabled: true,
    analyzer: 'pattern',
  }),
)(defineConfig({
  cli: 'claude',
  maxIterations: 50,
}))
