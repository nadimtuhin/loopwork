Backends API

Import

typescript 
import { 
  withJSONBackend, 
  withGitHubBackend,
  withNotionBackend 
} from 'loopwork/backends'

withJSONBackend Options

Option Type Default Description
tasksFile string '.specs/tasks/tasks.json' Path to tasks JSON file
tasksDir string '.specs/tasks' Directory for task descriptions

withGitHubBackend Options

Option Type Required Description
repo string Yes Repository (owner/repo)
label string No Filter by label
state 'open' | 'closed' | 'all' No Issue state filter

CLI Commands

Command Description
loopwork Run the task automation loop
loopwork --dry-run Preview tasks without execution
loopwork --resume Resume from saved state
loopwork --reset Reset all task states
loopwork --status Show current task status
loopwork --debug Run with debug output
loopwork --version Show version

TypeScript Types

Task Interface

typescript 
interface Task {
  id: string;
  title?: string;
  description?: string;
  status: 'pending' | 'in-progress' | 'completed' | 'failed';
  priority?: 'low' | 'medium' | 'high';
  dependencies?: string[];
  parentId?: string;
  metadata?: Record<string, any>;
  createdAt?: Date;
  updatedAt?: Date;
}
