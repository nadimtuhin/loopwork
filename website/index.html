CAPABILITIES

FEATURES

Powerful automation with cybernetic precision

Multiple AI Backends

Support for Claude, OpenCode, and Gemini with automatic failover between models on failure.

Plugin Architecture

Next.js-style composable plugins for Telegram, Discord, Asana, Todoist, Everhour, and more.

Real-time Execution

Watch AI models work through your task backlog in real-time with streaming output.

Smart Retries

Automatic failover between AI models with rate limit handling and timeout management.

Cost Tracking

Monitor token usage and enforce daily budgets to keep AI spending under control.

Rich Notifications

Get updates via Telegram, Discord, or Slack when tasks complete or fail.

Task Sources

Flexible backends - use GitHub Issues, JSON files, or build custom task sources.

Sub-tasks & Dependencies

Hierarchical task structures with dependency management and automatic ordering.

INSTALLATION

QUICK START

# Clone the repository
git clone https://github.com/nadimtuhin/loopwork.git
cd loopwork

# Install dependencies
bun install

# Build Loopwork
bun run build
// loopwork.config.ts
import { compose, defineConfig } from 'loopwork'
import { withJSONBackend } from 'loopwork/backends'

export default compose(
  withJSONBackend({
    tasksFile: '.specs/tasks/tasks.json'
  }),
)(defineConfig({
  cli: 'claude',
  maxIterations: 50
}))
# Navigate to example
cd examples/basic-json-backend

# Run tasks
./quick-start.sh --run

# Or use CLI directly
loopwork
