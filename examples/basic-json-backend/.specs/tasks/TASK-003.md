## Goal
Write a simple README.md file for the project

## Requirements

- Add a brief description

- Show usage examples

```markdown
A simple TypeScript project with basic utility functions.

### sayHello()

### sum(a, b)
Adds two numbers together.

## Usage

`npm install`
`npm test`
```
